# 用户管理

切换到 admin 数据库
```js
use admin
```

```js
// 返回所有用户
db.getUsers()
```


```js
db.createUser({
  user: "admin",    // 用户名
  pwd: "123",       // 密码（生产环境禁止用简单密码！）
  roles: [
    { role: "root", db: "admin" }  // root 角色（超级管理员），归属 admin 库
  ]
})
```
