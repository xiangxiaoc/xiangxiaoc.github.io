# 内置变量

| 变量      | 含义                     |
|---------|------------------------|
| $0		    | 脚本名字                   |
| $1		    | 位置参数 #1                |
| $2 - $9	 | 位置参数 #2 - #9           |
| $\{10}	 | 位置参数 #10               |
| $#		    | 位置参数的个数                |
| $*		    | 所有的位置参数(作为单个字符串) *     |
| $@		    | 所有的位置参数(每个都作为独立的字符串)   |
| ${#*}	  | 传递到脚本中的命令行参数的个数        |
| ${#@}	  | 传递到脚本中的命令行参数的个数        |
| $?		    | 返回值                    |
| $$		    | 脚本的进程ID(PID)           |
| $-		    | 传递到脚本中的标志(使用set)       |
| $_		    | 之前命令的最后一个参数            |
| $!		    | 运行在后台的最后一个作业的进程ID(PID) |
